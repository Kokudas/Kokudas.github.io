StageB v0.1
— S 확률 계산
홈 StageA Index
보유 펫의 실제 1레벨 표기 정수치. 샴기르/해피 계산기와 동일 입력.
용어 간단 설명
• 성공 확률 = 평균 p(S|Int) : 입력한 정수 초기치(Int)를 만족한다고 가정했을 때, Lv.140에서 환각(S) 달성 확률의 후보-가중 평균입니다. 최종 출력은 이 값 하나만 크게 표기합니다.
• p(S|cand) (내부) : 특정 후보(cand)가 사실일 때의 무조건적 S확률. UI에 표시하지 않습니다.
• p(Int|cand) (내부) : 해당 후보가 입력 정수(Int)를 만들 수 있는 확률. 후보 가중치 산정에만 사용됩니다.
