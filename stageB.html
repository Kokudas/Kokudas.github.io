단계B – S랭크 확률

홈 단계A 이동 단계B 이동

⚠️ 주의: 계산 결과는 실제 게임과 다를 수 있으며, 학습·참고용입니다. 현재 버전은 최대 140레벨 누적 성장을 기준으로 S랭크 달성 확률을 추정합니다.
후보는 JSON 파일로 저장·불러오기 하거나, 로컬 스토리지(localStorage)에서 자동 불러오기 됩니다. 페트 이름별로 관리 가능합니다.

입력

옵션

요약

# 랭크 base[체,공,방,순] initnum p(S|cand)

설명 (요약)

• 단계A에서 후보를 저장하면 localStorage에 JSON이 자동 저장됩니다.
• 단계B에서는 같은 페트 이름을 입력하면 자동으로 불러옵니다.
• 필요시 수동으로 JSON 파일을 불러올 수도 있습니다.
• 각 후보에 대해 140레벨 환각 달성 확률을 추정하고, 평균/범위를 표시합니다.
