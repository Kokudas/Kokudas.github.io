스톤에이지 S랭크 성장 확률 계산기

펫을 검색하고, 정수 표기 초기치를 입력하면, 모형(K=200, A~조합균등, B~Rank범위 균등) 가정하에 S 랭크 달성 확률을 추정합니다.

① 펫 선택 또는 수동 입력

자동완성에서 선택하거나, 아래 드롭다운을 사용하세요.
HP=4v+s+t+d • ATK=0.1v+s+0.1t+0.05d • DEF=0.1v+0.1s+t+0.05d • DEX=d
즐겨찾기(☆)로 자주 쓰는 펫만 보기 필터 가능

② S 랭크 기준 & 시뮬레이션

프리셋:
모형: K=200, A는 합10의 조합 균등, B는 Rank 범위 균등 샘플.

③ 결과

S 달성 확률 (4스탯 AND)
-
체/공/방/순 개별 확률
-
추정 Rank / B범위
-
추정 base(체/공/방/순)
-
항목값

관리자 영역

암호가 올바르면 아래 입력 폼이 열립니다.

펫 모델 값 입력

이름C0초기치(HP/ATK/DEF/DEX)성장률(HP/ATK/DEF/DEX)관리
모형/공식은 사용자가 제공한 설명을 따릅니다. 실제 게임 서버/버전에 따라 차이가 있을 수 있습니다.
