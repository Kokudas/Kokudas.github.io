펫 계산기 가이드

샴기르/해피 스타일과 유사한 흐름으로 구성되었습니다.

빠른 이동

StageA (후보 역산) StageB (S 확률)

사용 절차

  1. StageA에서 표기 초기치(소수 둘째)와 표기 성장률(소수 둘째) 입력 → 후보 역산.
  2. StageA에서 “StageB로 저장(로컬)” 클릭.
  3. StageB에서 펫 선택 + 초기치 정수 입력 → “계산 시작”.
  4. 크게 표시되는 “성공 확률 = 평균 p(S|Int)”만 보면 됩니다.

주의 / 안내

  • 라운딩 규칙: Floor(버림) vs Round(반올림). 서버 표기에 맞춰 선택하세요.
  • 결과는 참고용입니다. 내부 가정/편차로 오차가 발생할 수 있습니다.
  • 브라우저 로컬스토리지 사용. StageB의 “StageA 값 초기화”로 삭제 가능.
