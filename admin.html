미선택 —
정석 표기치 (소수 둘째, 표기는 3째자리 반올림)
S0 초기치 (체/공/방/순)
Sg 성장치 (체/공/방/순)
Nova2 e 후보
k/u/e 추정
SG 오차(합)
추정 k
추정 u (체/공/방/순)
선택 e
※ Nova2 호환: SG는 e/10000 스케일, S0는 k/100 스케일로 분리하여 u 정수(±2.5 보정) 복원 후, k∈[16..40]에서 표기식 완전일치(floor) 우선 선택.
JSON 스키마 — 예시: 
{
  "카르곤": {
    "s0": { "hp": 51.35, "atk": 10.85, "def": 6.64, "agi": 8.97 },
    "sg": { "hp":  9.78, "atk":  2.07, "def": 1.26, "agi": 1.71 },
    "k": 23,
    "u": { "hp": 31, "atk": 40, "def": 20, "agi": 39 },
    "e": 515,
    "s0_l1": 0,
    "sg_err": 0.02,
    "updated": 1730000000000
  }
}
↳ 필수: s0 (가능하면 sg)
↳ 추정: k, u, e는 Nova2 방식으로 자동 산출 가능.
기본 저장소: LocalStorage. 배포용은 JSON 내보내기 후 저장소에 커밋하세요. 원격 불러오기를 누르면 pets.json을 가져와 로컬에 캐시합니다.
