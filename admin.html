JSON 스키마 안내 — 예시: 
{
  "카르곤": {
    "s0": { "hp": 51.35, "atk": 10.85, "def": 6.64, "agi": 8.97 },
    "sg": { "hp":  9.78, "atk":  2.07, "def": 1.26, "agi": 1.71 },
    "k": 26,
    "u": { "hp": 23, "atk": 31, "def": 13, "agi": 30 },
    "mape": 0.12,
    "r": "535",
    "updated": 1730000000000
  }
}
↳ 필수: s0 (가능하면 sg도)
↳ 자동: k, u, mape는 저장 시 S0로 역추정 시도. 임계 초과 시 자동추정은 보류됨.
